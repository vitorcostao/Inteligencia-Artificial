# Exercício 01

Uma universidade quer prever se um aluno tem alta
chance de abandonar o curso no próximo semestre
para que possa intervir antecipadamente.

- Qual a tarefa?

    Se a hipótese será determinar um rótulo a partir de dados(faltas e notas), logo a tarefa é classificação.

- Possíveis atributos de entrada(features)?

    Frequência nas aulas, notas e entre outros.

- Existe atributos de saída(alvo/target)? Qual?

    Os atributos de saída são: Evasão e Permanência.

# Exercício 02

Uma empresa de bicicletas compartilhadas quer prever
quantas bicicletas serão alugadas na Lagoa da
Pampulha ao longo do dia.

- Qual a tarefa?

    Se a hipótese será determinar valores a partir de dados, logo a tarefa é regressão.

- Possíveis atributos de entrada(features)?

    Previsão do tempo, surtos de doenças, horários de pico.

- Existe atributos de saída(alvo/target)? Qual?

    O número de bicicletas alugadas.

# Exercício 03

Um hospital quer descobrir quais medicamentos
costumam ser prescritos juntos para determinados
perfis de pacientes.


- Qual a tarefa?

    Se a hipótese é determinar algo com base em padrões frequentes, a tarefa é associação

- Possíveis atributos de entrada(features)?

    Idade do paciente, peso do paciente, registro de doenças hereditárias.

- Existe atributos de saída(alvo/target)? Qual?

    Não há atributos de saída, apenas uma relação estabelecida entre instâncias.
